## Quick Start for Developers

1. **Backend**: Navigate to `services/api`, run `npm install` and `npm run dev`.
2. **Mobile**: Navigate to `apps/mobile`, run `npm install` and `npx expo start`.
3. **Database**: Ensure you have a Supabase project and local migrations are applied.
